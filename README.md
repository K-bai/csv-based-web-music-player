# CSV Based Web Music Player
以csv文件作为数据库的纯前端音乐播放器，使用vue-cli进行构建


## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
